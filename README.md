# Serverless Inference Metrics Simulation

This Python script simulates serverless inference requests, generating key performance metrics like total duration, inference duration, and error rate. It helps understand how to measure and analyze the performance of serverless AI model deployments.

## Language

`python`

## How to Run

Run this script using Python: `python serverless_metrics.py`
